# URL Shortener

This project is super simple, this is a kind of recreation of the same concept as Tiny Url!

## Architecture

### Front End

1. React
2. Context API
3. React Router Dom
4. Styled Components

### Front End Testing

1. Cypress - E2E
2. Jest + React Testing Library - Unit / Integration

### Back End

1. Node.js / Express
2. PostgreSQL + Sequelize

### Back End Testing

1. Mocha + Chai - Integration
